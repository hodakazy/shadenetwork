// About the Network

WHAT IS SHADE?

No land claiming. No teleporting. Real consequences. Shade Network is Minecraft the way it was meant to be played.

🌐

The Network

Built under the Look'around brand, running on Paper 1.21.11 with cross-version support back to 1.16.

  • Vanilla, Fabric, Forge, OptiFine — all clients welcome
  • Cross-version: 1.16 to 1.21.11
  • Built-in economy with structured shop
⚔️

The Philosophy

Risk is intentional. Your base can be raided. You walk your own distances. Every interaction carries weight.

  • No land claiming or TPA features
  • Real consequences for every decision
  • New SMP chapter every 1–2 months
📋

The Community

Rules, sell lists, staff applications, and announcements all live in Discord — your off-server home base.

  • Type /rules in-game or read in Discord
  • Staff applications via Discord ticket
  • Earn money selling items or bounty hunting

// How to Join

GET IN THE GAME

Four steps and you're in.

01

Get Minecraft Java Edition

You'll need Minecraft Java Edition. Any version from 1.16 to 1.21.11 works, but we strongly recommend 1.21.11 for the best experience.

02

Choose Your Client

All clients are supported — Vanilla, Fabric, Forge, OptiFine, and more. Use whatever you're already on.

03

Add the Server IP

Open Minecraft → Multiplayer → Add Server. Paste the IP from the panel on the right, save, and connect.

04

Know the Rules

Run /rules in chat or read them in Discord before you play. Ignorance isn't an excuse on Shade Network.
